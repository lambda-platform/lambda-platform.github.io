Эхлэх Танилцуулга бичлэг

Хүчирхэг боломжууд

Lambda Platform нь дараах хэд хэдэн модулиудтай бөгөөд эдгээр нь хөгжүүлэгч нарт програм хангамжийг ухаалгаар хурдан өндөр бүтээмжтэй гүйцэтгэх ДАВУУ ТАЛЫГ олгоно.

Lambda admin app

4-н төрлийн цэс, 100% responsive, гэрэлтэй болон харанхуй горим, олон өнгөний сонголт

Өгөгдөл бүртгэх орчин

Маягт, Хүснэгт, Маягт хүснэгтийн нэгтгэл, Нэмэлт үйлдлүүд (From, grid Trigger)

Хэрэглэгч, хандах эрх

Хэрэглэгч, Хандах эрх, зөвшөөрөл, Цэсний тохиргоо, Хандах эрхийн хязгаарлалт

GraphQL API

Нэмэх, засах, устгах, Хайлт, эрэмбэлэлт, хуудаслалт, Баримт бичиг, Нууцлал

Мэдэгдэл

Гар утас
