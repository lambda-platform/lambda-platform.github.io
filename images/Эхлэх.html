Lambda platform

Lambda Platform-г ашиглан энгийн бүртгэлийн веб сайтын админаас ERP систем хүртэл төрөл бүрийн системийн хөжгүүлэлтийг хийх боломжтой. Мөн төслөө Microservice бүтэцтэйгээр зохион байгуулан өндөр ачаалал авдаг том хэмжээний системийг хүртэл хийж болно.

Back-End

Go (Fiber), PHP (Laravel) гэсэн төрлийн back-end хэлдээр сонгон ашиглах боломжтой бөгөөд

одоогоор Graphql зөвхөн Go хэл дээр байгаа

Front-End

Admin app Front-End

Vue 3, Nuxt, Ant Design Vue, SASS, Tailwind CSS ашиглан хийсэн. Нэвтрэх хэсэг, хувийн мэлээлэл, нууц үг солих, 4-н төрлийн цэс, маягт болон хүснэгттэй хуудас агуулсан, 100% responsive, гэрэлтэй болон харанхуй горим, олон өнгөний сонголт бүхий бүтэн бүртгэлйин систем юм. Admin app front-end GitHub repo

Lambda нь MySQL, PostgreSQL, SQL Server, Oracle өгөгдлийн сангуудыг дэмждэг.
