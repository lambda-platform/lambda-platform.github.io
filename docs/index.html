Lambda Platform

Lambda Platform-г ашиглан энгийн бүртгэлийн веб сайтын админаас ERP систем хүртэл төрөл бүрийн системийн хөгжүүлэлтийг хийх боломжтой. Мөн төслөө Microservice бүтэцтэйгээр зохион байгуулан өндөр ачаалал авдаг том хэмжээний системийг хүртэл хийж болно.

Lambda Starter App

Go, PHP гэсэн 2 төрлийн back-end хэл дээр хөгжүүлэгдсэн.

Боломжууд

  • Нэвтрэх хэсэг
  • Хувийн мэдээлэл
  • Нууц үг солих
  • 4-н төрлийн цэс
  • Маягт болон хүснэгтийн 6-н төрлийн загвар
  • 100% responsive
  • Гэрэлтэй болон харанхуй горим (Vue 3)
  • Олон өнгөний сонголт (Vue 3)
  • Data Form
  • Data Grid
  • Krud
  • Notify
  • Agent
  • CRUD Logger
  • Datasource
  • OAuth2 server (Go хэл дээр)
  • Graphql (Go хэл дээр)
