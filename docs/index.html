Lambda Platform

Lambda Platform-г ашиглан энгийн бүртгэлийн веб сайтын админаас ERP систем хүртэл төрөл бүрийн системийн хөжгүүлэлтийг хийх боломжтой. Мөн төслөө Microservice бүтэцтэйгээр зохион байгуулан өндөр ачаалал авдаг том хэмжээний системийг хүртэл хийж болно.

Back-End

Go, PHP гэсэн 2 төрлийн back-end хэлдээр сонгон ашиглах боломжтой.

Back-End Модулиуд

  • Data Form
  • Data Grid
  • Krud
  • Notify
  • Agent
  • CRUD Logger
  • Datasource
  • OAuth2 server (Go хэл дээр)
  • Graphql (Go хэл дээр)

Lambda Go

Go хувилбарыг Fiber Framework дээр тулгуурлан хийсэн Go package github.com/lambda-platform/lambda бөгөөд модиулуудыг дурын Go төсөл дотор салган ашиглах боломжтой.

Дэмждэг өгөдлийн сангууд: MySQL, PostgreSQL, SQL Server, Oracle

Lambda Go GitHub repo

Lambda PHP

PHP хувилбарыг Laravel framework дээр тулгуурлан хийсэн Packagist package

Lambda PHP GitHub repo

Laravel Starter App GitHub repo

Дэмждэг өгөдлийн сангууд: MySQL, PostgreSQL, SQL Server

Front-End

Front-End нь Vue 2, Vue 3, React JS дээр ашиглах хувилбаруудтай

Front-End Модулиуд

  • Data Form
  • Data Grid
  • Krud
  • Notify
  • Agent
  • Page
    Гэх модулиудыг агуулсан npm package @lambda-platform/lambda-vue, модиулуудыг дурын Vue JS project дотор салган ашиглах боломжтой. Front-End GitHub repo

Admin App

Go Fiber framework дээр Go, Vue 3 хувилбаруудыг суулган Back-End болон Front-End-г нэгтгэж бэлдлсэн эхлүүлэх төсөл юм.

Admin App Боломжууд

  • Нэвтрэх хэсэг
  • Хувийн мэлээлэл
  • Нууц үг солих
  • 4-н төрлийн цэс
  • Маягт болон хүснэгтийн 6-н төрлийн загвар
  • 100% responsive
  • Гэрэлтэй болон харанхуй горим
  • Олон өнгөний сонголт

Admin App Front-End
