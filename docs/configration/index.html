Тохиргоо

Үндсэн тохиргоо

Тухайн төслийн үндсэн тохиргоо .env Үндсэн тохиргоо

Тохиргооны талбарТайлбар
DB_CONNECTIONӨгөгдлийн сангийн холболтын төрөл (mssql, postgres, mysql, oracle)
DB_HOSTӨгөгдлийн сангийн серверийн хаяг
DB_PORTӨгөгдлийн сангийн порт
DB_DATABASEӨгөгдлийн сангийн нэр
DB_USERNAMEӨгөгдлийн сангийн хэрэглэгчийн нэр
DB_PASSWORDӨгөгдлийн сангийн хэрэглэгчийн нууц үг
DB_DEBUGӨгөгдлийн сангийн алдааны мэдээллийг харуулах эсэх (true - харуулах, false - түгжих)
APP_NAMEСистем нэр
APP_PORTСистем ажиллах порт
APP_MIGRATEӨгөгдлийн сангийн хүснэгтийг автоматаар үүсгэх эсэх (true - үүсгэх, false - болих)
APP_SEEDСистемийн анхны өгөгдлийг өгөгдлийн санд оруулах эсэх (true - оруулах, false - болих)
SYSADMIN_LOGINСистем администраторын нэвтрэх нэр
SYSADMIN_PASSWORDСистем администраторын нууц үг
SYSADMIN_EMAILСистем администраторын имэйл хаяг
JWT_SECRETJWT хамгаалах нууц түлхүүр
JWT_TTLJWT токены хугацаа (цагаар)
MAIL_MAILERИ-мэйл илгээх аргын төрөл (жишээ: smtp)
MAIL_HOSTИ-мэйл серверийн хост хаяг
MAIL_PORTИ-мэйл серверийн порт
MAIL_USERNAMEИ-мэйл серверийн хэрэглэгчийн нэр
MAIL_PASSWORDИ-мэйл серверийн хэрэглэгчийн нууц

logo_squire_darkХаранхуй горимын дөрвөлжин логоны зам
logoTextЛогоны текст
super_urlСупер хэрэглэгчийн URL
app_urlАппын URL
has_languageХэлний төлөв

krud_publicКрудын олон нийтийн төлөв
languagesХэлний жагсаалт

role-redirectsХандах эрхийн чиглүүлэх холбоос
user_data_fieldsХэрэглэгчийн мэдээллийн талбарууд
data_form_custom_elementsData form Нэмэлт элементүүд
data_grid_custom_elementsData grid Нэмэлт элементүүд
password_reset_time_outНууц үгийг сэргээх хугацаа
static_wordsСтатик үгс
notifyМэдэгдэлийн тохиргоо Firebase config
custom_iconsНэмэлт icons
